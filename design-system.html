3) Layout Rhythm

  • Container width: 1240px max

  • Grid: 12-like flexible with 2/3/4 columns
  • Large margins inspired by luxury brand systems

5) RTL Rules

  • Direction: rtl globally with logical CSS properties.
  • Text alignment and spacing verified in cards, nav, forms, and tables.
  • English labels remain readable without breaking Arabic flow.

6) Accessibility & Motion

  • WCAG AA-oriented contrast pairs.
  • Keyboard focus states for links, buttons, and form controls.
  • Reduced-motion support for reveal/counter/carousel interactions.
  • Semantic structure and alt-text-ready image system.
